# Lidl Store Sales Analysis

This repository contains a Power BI project that provides a detailed analysis of sales and profit for a hypothetical Lidl store. The project leverages various Power BI visualizations and analytics tools to uncover insights from the dataset.

## Overview

The analysis focuses on several key areas:
- **Sales Overview**: Total sales and their breakdown by category, segment, and time.
- **Profit Analysis**: Profit margins across various categories and sub-categories.
- **Time-Based Insights**: Trends in sales and profit over months, quarters, and years.
- **Geographical Insights**: Sales performance across different countries.
- **Key Performance Indicators (KPIs)**: Highlighting total sales, total profit, and other critical metrics.
- **Interactive Exploration**: Drill-down capabilities using decomposition trees and Q&A features.

## Dataset

The dataset used in this project is **fabricated** and not based on real-life data. It was created solely for educational and demonstration purposes to showcase the capabilities of Power BI in analyzing and visualizing sales data. Any resemblance to actual data or entities is purely coincidental.

## Features

- **Sales by Segment**: Analyzes sales and profit by consumer, corporate, and home office segments.
- **Category Insights**: Detailed breakdowns by product categories like furniture, technology, and office supplies.
- **Monthly and Quarterly Trends**: Evaluates seasonal trends and variations in sales and profit.
- **Profitability Analysis**: Examines profit margins for categories and sub-categories.
- **Top and Bottom Performers**: Highlights the best and worst-performing products, regions, and sales representatives.

## Getting Started

### Prerequisites
To explore the Power BI report, you need:
1. [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. Clone this repository to your local machine.

### Usage
1. Open the `.pbix` file in Power BI Desktop.
2. Explore the interactive dashboards and visualizations.

## Visualization Highlights

- **KPI Metrics**: Quick overview of sales, profit, and cost.
- **Decomposition Tree**: Drill down into profitability drivers by category, sub-category, and region.
- **Time-Based Charts**: Identify trends over time for better business insights.
- **Geographical Breakdown**: Evaluate regional performance using maps.

## Disclaimer

The data used in this project is entirely fictional and was created for demonstration purposes. It should not be interpreted as real or representative of actual Lidl store sales.
