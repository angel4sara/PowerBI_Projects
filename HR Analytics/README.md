# HR Analytics Project

This repository contains a comprehensive Power BI dashboard and analysis focused on HR Analytics. The project leverages employee data to uncover insights about attrition, demographics, and job satisfaction. It is designed to help organizations better understand workforce dynamics and make informed decisions.

## Project Overview

The **HR Analytics Dashboard** presents insights derived from HR data, including:

- **Employee Attrition**:
  - Total employees: 1,470
  - Total attrition: 237 employees (16.1% attrition rate)

- **Key Metrics**:
  - Average age: 37 years
  - Average salary: $6.5K
  - Average years of employment: 7.0 years

- **Insights by Segments**:
  - Attrition by Education
  - Attrition by Age
  - Attrition by Salary Brackets
  - Attrition by Job Roles
  - Attrition by Employment Duration

- **Job Satisfaction**:
  - Satisfaction levels broken down by roles.

## Dashboard Features

The Power BI dashboard provides an interactive and intuitive interface to explore:

1. **Demographic Trends**:
   - Gender distribution
   - Age groups most affected by attrition

2. **Role-Based Insights**:
   - Attrition rates and satisfaction levels segmented by job roles like Healthcare Representatives, Research Scientists, Sales Executives, and more.

3. **Salary Trends**:
   - Analysis of attrition based on salary brackets.

4. **Employment Duration**:
   - Attrition across different employment tenure groups.

## Key Visualizations

The project uses a variety of Power BI visualizations to provide a clear understanding of the data:
- Bar and column charts for attrition by job role, salary, and age group.
- Pie charts for gender and education distribution.
- Tabular breakdowns of job satisfaction levels.

## Data Source

The data used in this project is simulated HR data and is intended for educational and analytical purposes only.
